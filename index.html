🚀 步步教學 (Step-by-Step)

1

準備 GitHub 戶口

GitHub 係一個放程式碼嘅「雲端夾萬」。

  • 去 GitHub.com 註冊。
  • 開一個新嘅 Repository (簡稱 Repo，即係一個項目文件夾)。
  • 名可以隨便起，例如 my-ai-app。
  • 揀 Private (私人) 就最安全。
2

由 AI Studio 攞 Code

喺 Google AI Studio 整好你個 AI 之後：

  1. 撳右上角個 "Get Code" 掣。
  2. 你會見到好多 code，唔使驚。揀 "Download" 攞 ZIP 檔。
  3. 解壓個 ZIP 檔，入面所有檔案 Drag 入去你頭先開嗰個 GitHub Repo 度。
3

最重要：改 Environment Variables

如果你直接將 API Key 寫喺 Code 入面，條 Key 會失效或者俾人偷！

喺你下載嘅檔案入面（通常係 index.html 或者 main.js），搵到 API Key 嗰行，將佢改做：

const apiKey = import.meta.env.VITE_GEMINI_API_KEY;

* 註：如果你係普通 HTML，呢度要視乎你點寫。唔識改嘅話可以問我！

4

喺 Vercel 開波

最後衝刺：

  1. 去 Vercel.com 用 GitHub 戶口 Login。
  2. 撳 "Add New" -> "Project"。
  3. 搵返你個 Repo 撳 Import。
  4. 喺 Environment Variables 嗰欄，Name 填 VITE_GEMINI_API_KEY，Value 貼返你條真嘅 API Key。
  5. 撳 Deploy，等 1 分鐘就收得工！

❌ 常見錯誤 (Common Errors)

404 Not Found: 通常係因為你個主檔案唔係叫 index.html。
API Key Invalid: 檢查下喺 Vercel 設定入面有無打錯字，或者有無多咗個空格。

❓ 常見問題 (FAQ)

Q: Vercel 同 Netlify 揀邊個？

A: 兩個都好！Vercel 部署速更快，UI 比較現代；Netlify 就功能比較多一啲。新手推薦先用 Vercel。

Q: 如果我想改 Code，係咪要重新 Deploy？

A: 唔使！只要你將新 Code 推上 GitHub，Vercel 會自動偵測到並幫你自動更新個網，超方便！
